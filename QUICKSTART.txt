Wi-Fi Phone Control - Quick Start

System requirements:
- Windows 10/11 x64
- Android wireless debugging enabled
- PC and Android device on a reachable network

Release usage:
1) Download WiFiPhoneControl-v1.0.0-win-x64.zip.
2) Extract the ZIP to a local folder.
3) Double-click WiFiPhoneControl.exe.
4) On the Android device, open:
   Developer options -> Wireless debugging -> Pair device with pairing code
5) In the app, fill Pair Endpoint and Pair Code.
6) Click Pair & Connect.
7) Click Start Control (scrcpy).

Bundled dependencies:
- scrcpy 4.0
- adb 37.0.0
- Full official scrcpy Windows x64 runtime under runtime\scrcpy

No separate scrcpy, adb, platform-tools, or winget install is required for release users.

Build from source:
1) Double-click build-release.cmd, or run .\build-release.cmd in PowerShell.
2) Output: release\WiFiPhoneControl-v1.0.0-win-x64.zip
3) The build script downloads and verifies scrcpy-win64-v4.0.zip automatically.
