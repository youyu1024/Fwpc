Wi-Fi Phone Control - Quick Start

1) Build release package
   - Double click: build-release.cmd
   - Output: dist\WiFiPhoneControl-v1.0.0-win-x64.zip

2) Release usage (recommended)
   - Extract dist\WiFiPhoneControl-v1.0.0-win-x64.zip
   - Double click: WiFiPhoneControl.exe
   - Runtime is already bundled (no winget needed)

3) First pairing
   - Device: Developer options -> Wireless debugging -> Pair device with pairing code
   - Fill Pair Endpoint + Pair Code
   - Click Pair & Connect

4) Daily use
   - Ensure Connected
   - Start Control (scrcpy)

5) UI features
   - Language: Chinese / English
   - Theme: Dark / Light
   - Profile list selection
   - Edit profile name on right, then Save Profile
   - Delete selected profile if needed

6) Build outputs
   - dist\WiFiPhoneControl.exe
   - dist\WiFiPhoneControl-v1.0.0-win-x64.zip

7) Source build requirement
   - Put full official scrcpy Windows package under runtime\scrcpy\ before building release
