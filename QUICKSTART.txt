Wi-Fi Phone Control - Quick Start

1) Build local dist package
   - Double click: build-release.cmd
   - Output:
     - dist\WiFiPhoneControl.exe
     - dist\WiFiPhoneControl-v1.0.0-win-x64.zip

2) Build publish release package (recommended)
   - Double click: package-release.cmd
   - Output:
     - release\WiFiPhoneControl-v1.0.0-win-x64.zip
   - If runtime\scrcpy\ is missing/incomplete, script will auto-provision from official scrcpy release.

3) Release user flow (fresh PC)
   - Download: WiFiPhoneControl-v1.0.0-win-x64.zip
   - Extract ZIP
   - Double click: WiFiPhoneControl.exe

4) First pairing
   - Device: Developer options -> Wireless debugging -> Pair device with pairing code
   - Fill Pair Endpoint + Pair Code
   - Click Pair & Connect

5) Daily use
   - Ensure Connected
   - Start Control (scrcpy)

6) Runtime completeness check
   - build-release.ps1 validates required runtime files:
     - scrcpy.exe
     - adb.exe
     - AdbWinApi.dll
     - AdbWinUsbApi.dll
     - SDL2.dll
